📊 Investment Analytics Dashboard (Power BI)
Overview

This Power BI dashboard analyzes investment portfolio performance using financial metrics such as NAV, CAGR, drawdown, and returns.

The dashboard provides interactive visualizations to track:

Net Asset Value (NAV) trends over time

Investment allocation distribution

Category-based investment analysis

Performance KPIs including 3M and 6M returns.

Features

Data cleaning and transformation using Power Query

Data modeling with relationships between tables

DAX measures for dynamic financial calculations

Interactive filters and slicers.
